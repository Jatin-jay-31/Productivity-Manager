# Notes Pro – Productivity Manager

A clean and responsive note-taking web application built to help users manage notes efficiently with authentication, search, sorting and theme support.

### Live Demo

[Project_Link](https://productivity-manager-6h4x-qkwm652k4-jatin-jay-31s-projects.vercel.app/)

### Features

* User authentication with Appwrite
* Create, edit and delete notes
* Search and sort notes
* Archive and Trash management
* Light and Dark theme support
* Responsive mobile-friendly UI
* Modal-based note creation flow
* Protected routes and user-specific data

### Tech Stack

* React.js
* Redux Toolkit
* React Router
* Tailwind CSS
* Appwrite
* Vercel

### Improvements & Challenges Solved

* Fixed authentication and session flow
* Improved mobile responsiveness
* Resolved deployment and CORS issues
* Added UI/UX polish and layout consistency
* Stabilized app flow and removed major bugs

### Project Goal

This project was built to simulate a real-world productivity application and strengthen frontend development, state management and backend integration skills.
